# CitadelQuest v0.7.2-beta — Update Notes (2026-03-04)

## New Feature: Citadel Explorer

The **Citadel Explorer** is a new tool integrated into the **CQ Contacts** page that lets you discover and interact with any public CitadelQuest profile — even before adding them as a contact.

### How to Use

1. Go to **CQ Contacts** (`/cq-contact`)
2. Enter a CitadelQuest profile URL in the Explorer input field (e.g. `https://example.citadelquest.com/username`)
3. Click **Explore** (or press Enter)

### What You Can See

The Explorer fetches and displays the public profile of any CitadelQuest user:

- **Profile photo** — displayed as a thumbnail for fast loading
- **Username and domain** — with a link to their public profile page
- **Bio** — with "show more" toggle for long bios (600+ characters)
- **Spirit showcase** — their AI Spirits with names, levels, XP, and colors
- **Shared items** — publicly shared files and CQ Memory Packs with content previews

### What You Can Do

- **Add Contact** — send a friend request directly from the Explorer
- **Download shared items** — download files and Memory Packs to your Citadel
- **Add to Library** — add Memory Packs to your CQ Memory libraries
- **View Memory Packs** — preview Memory Pack graphs inline

### Technical Details

- Profile data is fetched via a server-side proxy to handle CORS
- The Explorer URL input persists across page navigation (localStorage)
- Responsive layout — optimized for both desktop and mobile

## Profile Enhancements

### Public Profile Page

- **Full CQ Contact URL** displayed with a **copy-to-clipboard** button
- **Bio "show more" toggle** for bios longer than 600 characters
- **Thumbnail profile photos** (`.thumb`) served for faster loading

### CQ Profile Settings

- **No character limit** on Bio textarea — write as much as you want
- **Character count** displayed below the Bio field

### CQ Contact Detail Page

- **Bio "show more" toggle** for long bios (600+ characters)
- **Word-wrap** for bio text overflow

### CQ Contacts List

- **Bio text truncated** to 200 characters in contact cards for cleaner layout
- **Contact photos** fetched from remote Citadels
